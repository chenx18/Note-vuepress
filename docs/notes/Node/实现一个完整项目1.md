## 安装
1. koa脚手架： npm install -g koa-generator
2. 项目创建：  koa2 name
3. 安装中间件：npm install koa2-cors koa-jwt mysql md5 moment jsonwebtoken --save
4. 后端工具
```js
    koa
    koa-bodyparser 解析 PUT/POST 请求中的 body
    koa-convert
    koa-json
    koa-jwt jwt鉴权
    koa-logger
    koa-mysql-session
    koa-onerror
    koa-router
    koa-session-minimal
    koa-static
    koa-views
    koa2-cors 处理跨域
    md5 加密
    moment 时间处理
    mysql
```
